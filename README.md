# itau-credit-nfe-pipeline
Pipeline serverless AWS para validação de NFe, aplicação de regras de crédito e liberação de adiantamento, alinhado aos domínios U2, VH6 e SN do Itaú. Inclui arquitetura AWS (API Gateway → Lambda → DynamoDB → RDS → SQS), observabilidade com Datadog, CI/CD, testes automatizados, Clean Code, SOLID, DevOps e FinOps.
